# Data Structure Visualization Project

## Overview
Welcome to our Data Structure Visualization Project. This project is designed to bring core data structure concepts to life using the power of visual representation. Developed in C and utilizing the Raylib graphics library, our project provides an engaging, interactive way to understand and manipulate arrays. Our primary focus lies in array operations, including insertion, deletion, search, and insertion sorting.

## Features
- **Array Manipulation**: Interactive tools for performing basic array operations - insertion, deletion, and search.
- **Insertion Sort Visualization**: Watch how the insertion sort algorithm works in real time with our step-by-step visual guide.
- **Minimalistic Design**: Clean and simple UI/UX, ensuring an undistracted and focused learning environment.
- **User-Friendly Interface**: Intuitive controls and clear visual cues for effortless user interaction.
